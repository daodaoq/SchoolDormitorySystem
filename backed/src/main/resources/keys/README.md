# 支付宝沙箱密钥目录

请将从支付宝开放平台下载的密钥文件放置在此目录下。

## 需要准备的文件

| 文件名 | 说明 | 获取方式 |
|--------|------|----------|
| `alipay_private_key.pem` | 应用私钥（PKCS#8 格式） | 支付宝开放平台 → 开发服务 → 沙箱应用 → 开发信息 → 应用私钥 |
| `alipay_private_key_pkcs1.pem` | 应用私钥（PKCS#1 格式，可选） | 同上，如不需要 PKCS#1 格式可忽略 |
| `alipay_public_key.pem` | 应用公钥 | 与私钥配对生成，上传至支付宝开放平台 |

## 证书文件

以下证书文件放在 `src/main/resources/` 目录下（与此 keys 目录同级）：

| 文件名 | 说明 | 获取方式 |
|--------|------|----------|
| `appPublicCert.crt` | 应用公钥证书 | 支付宝开放平台 → 沙箱应用 → 开发信息 → 应用公钥证书 |
| `alipayPublicCert.crt` | 支付宝公钥证书 | 支付宝开放平台 → 沙箱应用 → 开发信息 → 支付宝公钥证书 |
| `alipayRootCert.crt` | 支付宝根证书 | 支付宝开放平台 → 沙箱应用 → 开发信息 → 支付宝根证书 |

## 注意事项

- ⚠️ **所有 `.pem` 和 `.crt` 文件已被 `.gitignore` 排除，不会被提交到 Git**
- 沙箱环境密钥仅用于开发测试，上线前请替换为正式环境密钥
- 私钥请妥善保管，不要通过任何方式泄露
