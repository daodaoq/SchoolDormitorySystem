# 学生宿舍管理制度

## 第一章 总则

**第一条** 为规范学生宿舍管理，保障学生住宿安全，维护正常生活秩序，特制定本制度。

**第二条** 本制度适用于本校所有在校住宿学生，包括本科生、研究生及留学生。

**第三条** 学生宿舍管理坚持"安全第一、服务为本、规范管理"的原则。

## 第二章 入住管理

### 2.1 入住条件

1. 本校正式注册的全日制在校学生
2. 身体健康，无传染性疾病
3. 已缴纳住宿费及相关费用
4. 签订《学生住宿协议书》

### 2.2 入住流程

1. **提交申请**：新生入学时在迎新系统提交住宿申请，老生在每学期末提交下学期住宿申请
2. **资格审核**：学生处对申请材料进行审核，3个工作日内反馈审核结果
3. **分配宿舍**：宿舍管理科根据专业、年级、性别等因素统一分配宿舍和床位
4. **缴纳费用**：通过学生宿舍收费管理系统在线缴纳住宿费（1200元/学期）及其他相关费用
5. **领取钥匙**：凭缴费凭证到各楼栋管理处领取宿舍钥匙和门禁卡
6. **入住登记**：填写《入住登记表》，确认宿舍设施完好情况

### 2.3 床位分配规则

- 原则上按同专业、同年级集中安排
- 预留少量床位供特殊需求调整
- 新生优先安排低楼层，高年级安排在较高楼层
- 留学生单独安排留学生公寓

## 第三章 宿舍调换

### 3.1 调换条件

学生有下列情况之一可申请调换宿舍：

1. 与室友存在严重矛盾，经辅导员调解无效的
2. 因身体原因（如过敏、残疾等）需调整住宿环境的
3. 因专业调整导致上课地点与宿舍距离过远的
4. 因宿舍设施严重损坏需整体搬迁的

### 3.2 调换流程

1. 学生向辅导员提交《宿舍调换申请表》
2. 辅导员核实情况并签署意见
3. 提交至宿舍管理科审核
4. 宿舍管理科查询空余床位，3个工作日内回复
5. 审批通过后，学生在5个工作日内完成搬离和迁入
6. 办理钥匙/门禁卡交换手续

## 第四章 退宿管理

### 4.1 正常退宿

- 毕业离校时统一办理退宿
- 离校前需完成宿舍验收（设施完好、卫生合格）
- 退还钥匙和门禁卡
- 结清所有费用

### 4.2 中途退宿

- 因病休学、退学、出国交流等情况可申请中途退宿
- 需提前15天提交书面申请
- 住宿费按实际住宿天数结算，多退少补

## 第五章 日常管理

### 5.1 作息时间

| 时段 | 时间 |
|------|------|
| 开门时间 | 06:00 |
| 关门时间 | 23:00（周日至周四）/ 23:30（周五、周六） |
| 熄灯时间 | 23:30（考试周延长至 00:30） |

### 5.2 访客管理

- 访客需在值班室登记，出示有效证件
- 访客时间：08:00 — 22:00
- 异性访客不得进入宿舍楼层，仅限一楼接待区
- 不得留宿外来人员，违者按校规处理

### 5.3 卫生管理

- 每个宿舍制定值日表，每日打扫
- 每周三下午为全校卫生检查日
- 每月进行文明宿舍评比

## 第六章 安全管理

1. 严禁使用大功率电器（电炉、热得快、电热毯等），功率上限为 800W
2. 严禁在宿舍内吸烟、使用明火
3. 严禁私拉乱接电线、改装电路
4. 离开宿舍时关闭电源、水源、门窗
5. 贵重物品自行妥善保管，丢失自负
6. 发现安全隐患及时报告宿舍管理员

## 第七章 违规处理

| 违规行为 | 处理措施 |
|---------|---------|
| 使用违规电器 | 没收电器 + 通报批评 + 扣 5 分 |
| 夜不归宿（累计 3 次） | 通报家长 + 取消住宿资格 |
| 留宿外人 | 严重警告 + 扣 10 分 |
| 破坏公共设施 | 照价赔偿 + 通报批评 |
| 宿舍卫生不合格（累计 3 次） | 通报批评 + 取消评优资格 |

扣分累计达到 15 分者，取消住宿资格，且不再安排校内住宿。

---

> 本制度自 2024 年 9 月 1 日起执行，解释权归学生宿舍管理科所有。
