# 宿舍调换与退费流程

## 一、宿舍调换

### 1.1 申请条件

满足以下任意一条可申请调换宿舍：

| 条件 | 所需材料 |
|------|---------|
| 室友矛盾经调解无效 | 辅导员调解记录 + 调换申请表 |
| 身体原因需特殊安排 | 医院证明 + 调换申请表 |
| 专业校区调整 | 学院证明 + 调换申请表 |
| 宿舍设施严重损坏 | 维修记录 + 调换申请表 |

### 1.2 申请流程

```
提交申请表 → 辅导员签字 → 宿舍管理科审核 → 查询空床位 → 审批 → 搬离/迁入
    ↓              ↓              ↓             ↓          ↓         ↓
  学生本人      3个工作日内     核实情况      数据库查询   3个工作日   5个工作日内完成
```

### 1.3 注意事项

- 调换申请每月 1—10 日集中受理，11—20 日为审核期，21 日起办理调换
- 每学期最多申请调换 1 次
- 调换后的宿舍费用按实际入住宿舍类型计算，多退少补
- 未经批准私自调换宿舍者，按违规处理，扣 10 分

## 二、退宿退费

### 2.1 退宿类型

| 类型 | 适用情况 | 费用结算方式 |
|------|---------|-------------|
| 毕业退宿 | 正常毕业离校 | 住宿费不退（已满整学期） |
| 休学退宿 | 因病或其他原因休学 | 按实际天数结算，多退少补 |
| 退学退宿 | 主动或被动退学 | 按实际天数结算，多退少补 |
| 出国退宿 | 学期中出国交流 | 按实际天数结算，多退少补 |
| 校外住宿 | 申请校外住宿获批 | 退回剩余学期的住宿费 |

### 2.2 退费计算规则

**住宿费退费**：

```
退费金额 = 已缴住宿费 − 已住宿天数 × 日住宿费

日住宿费 = 学期住宿费 ÷ 学期总天数（约 150 天）
```

举例：
- 标准四人间住宿费 1200 元/学期
- 日住宿费 = 1200 ÷ 150 = 8 元/天
- 如已住宿 30 天后退宿，退费 = 1200 − 30×8 = 960 元

**空调费退费**：
- 按剩余学期比例退费
- 如使用不满半学期，退 50%；超过半学期，不退

**网络费退费**：
- 网络费按月结算
- 已开通未满 1 个月，按 1 个月收费；超过 1 个月但不满 2 个月，按 2 个月收费

### 2.3 退费流程

1. **提交申请**：学生向宿舍管理科提交《退宿退费申请表》及相关证明材料
2. **宿舍验收**：管理员到宿舍检查设施完好情况，填写《宿舍验收单》
   - 设施损坏需照价赔偿，从退费中扣除
   - 卫生不合格需清理干净后再次验收
3. **费用核算**：财务处根据实际住宿天数核算退费金额
4. **审批**：宿舍管理科科长审批
5. **退款**：财务处在 3—5 个工作日内将款项原路退回至学生支付宝账户
6. **确认**：学生收到退款后在系统中确认

### 2.4 不予退费的情况

- 因违规被取消住宿资格的，不予退费
- 学期已超过 3/4 时间的，住宿费不予退还
- 未完成宿舍验收流程的，暂不退费

## 三、费用减免

### 3.1 减免类型

| 减免类型 | 减免比例 | 申请条件 |
|---------|---------|---------|
| 经济困难减免 | 最高 50% | 家庭经济困难认定 + 成绩合格 |
| 学业优秀减免 | 20% | GPA 排名专业前 5% |
| 勤工助学补贴 | 200 元/月 | 在宿舍管理科岗位服务 |
| 残疾学生减免 | 30% | 持有残疾证明 |

### 3.2 减免申请流程

1. 每学期开学前 30 天内提交减免申请
2. 提供相关证明材料（贫困证明、成绩单、残疾证等）
3. 学生资助管理中心审核
4. 审核通过后在缴费时自动抵扣

> 减免申请每年需重新提交，不可自动延续。
